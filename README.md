# NI-Mail-Agent
NI Mail Agent  Программа для проведения легальных массовых email рассылок с поддержкой socks-прокси и шаблонов автозамены текста (макросов)
- Программа для проведения легальных массовых email рассылок с поддержкой socks-прокси и шаблонов автозамены текста (макросов).

-  Можно рассылать как обычные письма, так и письма в формате HTML, можно прикреплять к письмам файлы. Рассылка достаточно большого количества писем возможна за счет использования в одном аккаунте = -  программы множества поочередно используемых почтовых ящиков на различных серверах. Рекомендуется для рассылки прайсов вашим партнерам, рассылки новостей посетителям вашего сайта, рассылки  - - приглашений на торжество вашим родственникам и друзьям и т.д. Бесплатна для домашнего некоммерческого использования. Для коммерческого использования необходимо бесплатно запросить серийный номер для - одноразовой активации редакции “Professional”.
